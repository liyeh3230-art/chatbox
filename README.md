## 🔥 啟用真實多人聊天

目前是演示模式，配置Firebase 後可實現真實的多人聊天！

### 簡易步驟：

1. **建立Firebase 專案**
   - 造訪：https://console.firebase.google.com
   - 建立專案→ 啟用Realtime Database

2. **取得配置**
   - 專案設定→ 註冊Web 應用
   - 複製firebaseConfig 程式碼

3. **更新index.html**
   - 搜尋`YOUR_API_KEY`
   - 替換為您的配置
   - 儲存文件

4. **設定資料庫規則**
   - Realtime Database → 規則
   - 貼上規則代碼（請參閱文件）
   - 發布

5. **測試**
   - 開啟兩個瀏覽器視窗
   - 應該能真實配對聊天！

**詳細教學請查看：📖 [Firebase設定教學.md](Firebase設定教學.md)**

## 🆘 常見問題

### Q: 配對失敗怎麼辦？
**A:** 檢查Firebase 設定是否正確，查看瀏覽器Console（F12）。

### Q: 演示模式和真實模式有什麼不同？
**A:** 演示模式是單機版，訊息是模擬的。真實模式需要配置Firebase，支援多人在線。

### Q: 免費版夠用嗎？
**A:** 對於個人專案和中小型網站完全夠用！ Firebase 免費額度可支援數千名日活用戶。

### Q: 如何保護使用者隱私？
**一個：**
- 不儲存個人資訊
- 匿名聊天
- 聊天記錄定期清理
- 建議使用HTTPS

### Q: 可以商用嗎？
**A:** 可以！ MIT 許可證允許商業使用。但請注意：
- 超出免費額度後Firebase 會收費
- 建議新增服務條款和隱私權政策
- 考慮加入內容審核機制
